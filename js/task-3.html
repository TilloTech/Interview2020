JavaScript Task 3 - Working with data

Task

  • Using XMLHttpRequest, jQuery, Axios, or the new Fetch API, fetch the orders data from this file: https://raw.githubusercontent.com/TilloTech/Interview2020/master/data/orders.json
  • Display the results in a table.
  • Bonus: Use the tablesorter jQuery plugin to make the table sortable by the customer's name with some indication as to whether it's in ascending or descending order.

Rules

  • Edit this file directly.
  • When you click the Run button:
    1. Programatically generate a table and add it as a child to the div with the ID "answer".
    2. For each order, create a row in the table with the order reference (UUID), price with currency symbol, customer's full name, customer's email address, and the a link to the URL with the Order ID as the link text.
    3. E.g. Using the first three orders from the file, it should look like the table below.
